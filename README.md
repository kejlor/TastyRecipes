# Tasty Recipes
This is an app where you can browse for new recipes by category. Apart from that you can browse for your desired meal by name. If application wont't find
your desired meal, you will get informed.
<br>
<br>
<br>

### Used Technologies:
1. SwiftUI
2. Unit Tests
3. SFSymbols
4. TheMealDB API
